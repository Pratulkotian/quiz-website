# 🧠 Quiz Website

A modern interactive quiz platform built using React, TypeScript, Tailwind CSS and Firebase.  
The application allows users to participate in quizzes with a clean and responsive interface.

## 🚀 Live Demo

https://quiz-website-eta-red.vercel.app


## ✨ Features

- User friendly quiz interface
- Dynamic question handling
- Responsive design
- Firebase backend integration
- Fast performance using Vite
- Clean modern UI


## 🛠️ Tech Stack

- React.js
- TypeScript
- Tailwind CSS
- Firebase
- Vite
- Vercel
